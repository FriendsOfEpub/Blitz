The Current State of Blitz

Published by Jiminy Panoz on February 8, 2020.

This page sums up what has been happening in Blitz recently. Please note it may not be up to date though.

For a global overview, please check The State of Blitz in 2019.

A Checklist Template

The companion apps have been sharing a lot of code, and it was high time to address this issue.

Blitz Checklist has been created to facilitate maintenance and updates in the existing Blitz Checklists, and was designed as a boilerplate anyone can use to create such web apps – with interactions, offline support, etc.

Blitz Checklist is using the Mustache logic-less template, which means data is now separated from markup, and should be easier to use in a non-web context as well.

And it has already proven very useful as it helped implement new features and bug fixes more quickly.

Every precaution has been taken in the migration to make sure this breaking change is workable for people who cloned or forked The ebook performance and ebook design checklists.

Design and Optim Apps Bumped to Version 2

The ebook performance and ebook design checklists have been bumped to version 2 following the creation of Blitz Checklist.

The following improvements have been rolled out:

  • Service Workers handle all critical files explictly;
  • checked boxes are handled when printing the checklist;
  • help instructions are a proper modal;
  • it’s possible to expand/collapse all details;
  • homescreen icon and splash screen are handled correctly on Android 8+.

We’ve designed this template in a way that makes Mustache (hence JSON) entirely optional. As long as you follow the templating rules in index.html, everything should be fine.

If you don’t want to migrate to Mustache and are more comfortable with the idea of manually maintaining the checklist, then you can use the v.1.0.4 tag in your fork – note you’ll have to backport fixes and improvements though, as there won’t be any further development for version 1.

We’ll be more than happy to help you create checklists for InDesign, Quark, and whatnot.

CSS Tricks App Bumped to Version 1

Finally, ebook Tricks should now be on par with the other companion apps. Unfortunately, Blitz ebook Tricks can’t leverage the Blitz Checklist template, but it’s been updated and bumped to version 1 nevertheless.

This new version brings:

  • copy CSS snippet to clipboard in one click;
  • smooth scroll for nav menu;
  • print CSS improvements;
  • better handling of homescreen icon and splash screen on Android 8+.

In the end, it doesn’t necessarily sound like much, but should make it a lot easier to use Blitz ebook Tricks in production.
